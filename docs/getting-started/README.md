# Getting Started with NetRaven

## Introduction

Welcome to the NetRaven Getting Started Guide. This section provides everything you need to begin using NetRaven for network configuration management. These guides are designed to help you install the system, complete initial setup, and start using the core features quickly.

## Core Getting Started Guides

### [Quick Start Guide](./quick-start.md)

The fastest path to get NetRaven running and managing your first network device:

- Install NetRaven using Docker Compose
- Add your first device
- Run your first configuration backup
- Navigate the main interface

**Time required:** Approximately 15-30 minutes

### [Installation Guide](./installation.md)

Comprehensive installation instructions for various deployment options:

- Docker-based installation (recommended)
- Manual installation process
- Database configuration options
- Storage configuration

**Time required:** Approximately 30-60 minutes

### [Initial Setup Guide](./initial-setup.md)

Essential configuration steps after installation:

- First login and password change
- System settings configuration
- Backup storage configuration
- Email notification setup
- Authentication integration
- User account creation
- Tag and rule configuration

**Time required:** Approximately 30-45 minutes

## Deployment Options

NetRaven supports multiple deployment scenarios:

- **Development Environment**: Single-host deployment for testing and development
- **Production Environment**: Scalable deployment for production use
- **High Availability Setup**: Redundant deployment for critical environments

For detailed deployment information, see the [Deployment Guide](../deployment/deployment-guide.md).

## System Requirements

Before installing NetRaven, ensure your environment meets these minimum requirements:

### Docker-based Deployment

- Docker Engine 19.03 or newer
- Docker Compose 1.27 or newer
- 4GB RAM minimum (8GB recommended)
- 20GB disk space
- Network connectivity to managed devices

### Manual Installation

- Linux-based operating system (Ubuntu 20.04 LTS or CentOS 8 recommended)
- Python 3.10 or newer
- PostgreSQL 12 or newer
- Node.js 14 or newer
- 4GB RAM minimum (8GB recommended)
- 20GB disk space
- Network connectivity to managed devices
